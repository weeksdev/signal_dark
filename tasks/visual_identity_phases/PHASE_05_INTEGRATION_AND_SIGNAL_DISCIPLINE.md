# Phase 05 - Integration And Signal Discipline

## Purpose

Unify the heavier, more material world with the Matrix-style signal treatment so the game still feels like itself.

This phase is where the team prevents the new weight from turning into generic sci-fi clutter.

## Primary Files Likely Involved

- `src/fx/StealthOverlay.gd`
- `src/world/Grid.gd`
- palette and color-control files
- any ship/enemy/environment assets touched in previous phases

## Deliverables

- harmonized visual stack
- verified signal language on top of solid forms
- final review batch
- checkpoint commit

## Microscopic Tasks

### 05.1 Layer Audit

- review floor, wall, ship, enemy, objective, UI, and post-process together
- identify any layers fighting for the same value range

### 05.2 Signal Restraint Pass

- ensure CRT, grain, scanlines, and phosphor treatment support the scene
- reduce any effect that erodes clarity or form
- preserve identity without washing the whole frame

### 05.3 Grid Discipline Pass

- ensure the grid remains secondary to actual world mass
- verify warp and anomalies feel smooth and intentional
- verify grid still contributes navigation and mood

### 05.4 Gameplay Priority Pass

- ship first
- immediate threat second
- world structure third
- ambient signal detail last

### 05.5 Cross-Scene Consistency Pass

- compare multiple rooms
- compare stealth and alert-adjacent situations
- compare objective and non-objective frames

### 05.6 Final Failure Scan

- any hollow-placeholder read remaining
- any area where the world collapses back into line art
- any visual effect that hides important stealth information
- any drift away from the stated pillar

## Snapshot Gate

Generate a final named review batch and compare against the baseline selected in Phase 00.

Required comparison questions:

1. Does the game now feel more like a top down stealth-action shooter?
2. Do actors and architecture feel materially distinct?
3. Does the Matrix-style signal aesthetic still define the experience?
4. Would an artist and gameplay designer both accept the current direction as a stronger foundation?

## Acceptance Criteria

- all major entity classes feel heavier and more intentional
- stealth readability remains intact
- environment, actors, and overlays feel like one visual system
- the mission statement is stronger than it was at baseline

## Commit Gate

Final checkpoint commit for the full phase track.
